# 🌦️ Weather App

## 📅 Date
17 March 2026

## 📌 Description
This is a simple Weather App built using React and Material UI.  
It shows weather details like temperature, humidity, and conditions.

## 🚀 Features
- Search city weather
- Display temperature, humidity
- Dynamic UI using Material UI
- Weather icons (hot, cold, rain)

## 🛠️ Tech Stack
- React.js
- Material UI
- JavaScript
- CSS
